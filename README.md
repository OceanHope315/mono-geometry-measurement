# 单目距离估计与多目标风险评估系统 (Mono Distance & Multi-Target TTC)

## 项目概述

本项目实现了一个基于单目视频的自动驾驶感知系统，能够实时检测车辆、估计距离、计算相对速度并评估碰撞时间（TTC）。系统结合了 YOLOv8 目标检测、MiDaS 深度估计以及鲁棒的目标跟踪算法。

### 主要特性

- **多目标检测与跟踪**：检测 `car`, `bus`, `truck`, `motorcycle` 等驾驶相关类别，并实现跨帧 ID 关联。
- **混合距离估计**：结合相机几何先验（focal length + height）与 MiDaS 深度图趋势修正。
- **鲁棒平滑处理**：对距离和速度采用中值过滤与异常值剔除，减少由于检测框抖动带来的噪声。
- **碰撞风险预警**：实时计算 TTC，并根据预设阈值（危险 2.0s / 警告 5.0s）动态调整检测框颜色。
- **批量自动化处理**：支持对目录下所有 MP4 视频进行一键式批处理。
- **全方位数据输出**：
  - **AVI 视频**：可视化处理结果。
  - **CSV 报表**：详细记录每帧每个目标的距离、速度和 TTC。
  - **多维度曲线图**：自动为每个视频中跟踪时间最长的 3 个目标生成 **距离、速度、TTC** 随时间变化的 3 类曲线图（10 个视频共生成 90 张图）。

## 文件结构

```
.
├── mono_distance_baseline.py      # 主程序（包含算法逻辑与批量处理入口）
├── requirements.txt               # 项目依赖列表
├── README.md                      # 本说明文件
├── videos/                        # 输入视频目录（存放待处理的 .mp4 视频）
└── results/                       # 自动生成的输出目录
    ├── avi/                       # 处理后的可视化视频 (.avi)

    ├── distance_curve/            # 目标距离变化曲线图 (.png)
    ├── speed_curve/               # 目标相对速度变化曲线图 (.png)

```

## 安装与配置

### 环境要求
- Python 3.8+
- NVIDIA GPU & CUDA (推荐，可大幅提升 MiDaS 深度估计速度)

### 快速开始

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **准备视频**
   将需要处理的 `.mp4` 视频放入根目录下的 `videos/` 文件夹中。

3. **运行程序**
   ```bash
   python mono_distance_baseline.py
   ```

## 输出详解

### 1. 多目标曲线图 (Results Visualization)
系统会自动识别每个视频中最重要的 3 个目标（基于跟踪稳定性），并在 `results/` 下的三个子目录中生成 90 张 PNG 图像（假设有 10 个输入视频）：
- 文件命名规则：`{视频名}_target_{ID}_{类型}.png`
- 距离图：展示目标与本车的实时米制距离。
- 速度图：展示目标的相对接近速度（正值代表接近，负值代表远离）。
- TTC 图：展示碰撞预估时间（无穷大代表无碰撞风险，已将 inf 剪切至 20s 以便观察趋势）。

### 2. 数据报表 (CSV Data)
CSV 文件记录了包含 `depth_score`（深度得分）、`distance_m`（平滑后距离）、`relative_speed_mps`（相对速度）和 `ttc_s`（碰撞时间）在内的核心数据。

## 技术细节

- **深度辅助**：MiDaS 模型用于提取目标的空间深度趋势，修正由于几何模型在不同高度目标上的误差。
- **速度计算**：采用 5 帧时间序列线性拟合，相比简单的位移差分更抗抖动。
- **风险等级**：
  - <font color="red">红色 (Danger)</font>: TTC < 2.0s
  - <font color="yellow">黄色 (Warning)</font>: TTC < 5.0s
  - <font color="green">绿色 (Safe)</font>: TTC >= 5.0s 或 目标正在远离
